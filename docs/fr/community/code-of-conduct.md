# Code de conduite

La communauté `piighost` accueille toute personne partageant l'objectif de rendre la protection des PII accessible et composable. Cette page résume les attentes de comportement. Pour la version complète qui fait autorité, voir [`CODE_OF_CONDUCT.md`](https://github.com/Athroniaeth/piighost/blob/master/CODE_OF_CONDUCT.md) à la racine du dépôt.

## Périmètre

Ce code s'applique à tous les espaces gérés par le projet :

- Issues, pull requests et discussions GitHub.
- Rencontres communautaires.
- Tout autre forum d'échange lié au projet.

## Comportements attendus

### Restez accueillants et patients

Accueillez les nouveaux contributeurs quel que soit leur niveau. Une question qui vous paraît basique peut être la première question de quelqu'un qui débute.

### Respectez les désaccords

Les choix techniques se discutent. Critiquez les idées, pas les personnes. Argumentez avec des exemples et des alternatives plutôt qu'avec des jugements de valeur.

### Signalez ce qui ne va pas

Si vous observez ou subissez un comportement inapproprié dans un espace du projet, contactez les mainteneurs par [email privé via le profil GitHub](https://github.com/Athroniaeth) ou via une [GitHub security advisory](https://github.com/Athroniaeth/piighost/security/advisories/new) si le sujet touche à la sécurité.

## Application

Les mainteneurs peuvent retirer, modifier, ou rejeter les contributions qui ne respectent pas ce code. En cas de violation répétée, les contributeurs peuvent être exclus temporairement ou définitivement des espaces du projet.
